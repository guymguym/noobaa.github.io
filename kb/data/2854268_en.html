NooBaa makes it simple to configure where will the data be written to. All you need to do it to set/edit the bucket data placement policy.

Steps:

  1. Go to Buckets > Bucket page > Edit Data Placement

  2. In “Bucket Data Placement Policy” window you can see the current configuration of this bucket data placement. Change the policy type or add/ remove resources according to your needs.

  3. Click “Save” when done editing.

  4. The storage calculations will be changed according to the new resources available storage. Such changes will happen in the background and data migration processing time depends on the amount of data in the bucket.
