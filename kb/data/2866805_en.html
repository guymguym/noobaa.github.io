Upgrading NooBaa is simple. Whenever a new version is available, an email will be sent over. 

If you missed the email, feel free to contact support@noobaa.com.

Once you have the latest upgrade package, please go to Management -> Version -> System Upgrade

Simply drag the upgrade package to the upgrade box or click on Browse button and select the upgrade package from its location.
