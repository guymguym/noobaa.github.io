Nodes pool is a group of storage nodes that can be used for a data placement policy within NooBaa's buckets. All assigned storage nodes in each pool will be used for capacity aggregation.

Steps:

  1. Go to Resources > Node Pools > Create Pool

  2. Type a valid pool name according to the name constraints

  3. Assign nodes to the new pool by moving them from other pools.
  • All currently used nodes will have to go through a migration process.
  • Each pool must have at least 3 nodes (for data replicas).

  • Make sure that the nodes you choose to move from another pool will not leave that pool with less than 3 nodes. If it does, that pool could no longer be available for data placement.
