Deployment of NooBaa BYOL image on Google cloud includes the following steps:
 
  1. Login to Google Cloud Console,
  2. Copy NooBaa image into your cloud storage by using google shell
           gsutil cp gs://<image link will be provided via email> gs://<your bucket name>

      3. Navigate to Compute Engine -> Images and click on Create Image button.





4. Create a new instance





5. tag the instance network 


 

5. Open the following ports:
  • 8080/8443 - management ports
  • 80/443 - S3 API compatible access ports
Unlike the following example, please make sure that you open the ports only for your application source IP, to reduce your risks.
