NooBaa is available in AWS marketplace as both Bring Your Own License (BYOL) and paid offers.

NooBaa BYOL offer requires an activation key. Apply for activation key by using the following link

NooBaa paid offer requires an IAM role with  AWSMarketplaceMeteringFullAccess policy and attached your EC2 an IAM role that will allow usage metering service.

1) The AWSMarketplaceMeteringFullAccess policy looks like that. Simply add it to the user that launches the AMI:

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Action": [
                "aws-marketplace:MeterUsage"
            ],
            "Effect": "Allow",
            "Resource": "*"
        }
    ]
}
2) Create a new IAM role and attach it to your EC2 instance
