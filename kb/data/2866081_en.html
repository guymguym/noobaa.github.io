Recommended OVA VM settings:
  • 8 CPUs, 16GB RAM or A3, A4 or A7 on Azure marketplace.
  • Three windows or Linux nodes, each with more than 15GB free space.
  • Alternatively, you can connect only cloud resource (Azure blob storage or AWS S3) and start working immediately.
     
Firewall
  • The nodes need network access on port 8443 to the VM
  • The nodes and the core VM need network access one to another on one of the ports 60100-60600 (configurable).
  • NooBaa core (VM) management ports are 8080 and 8443
  • High availability - open port 27000
  • S3 public access ports are 80 and 443
