Shutting down NooBaa core VM for any reason would be safer if done after putting the system in maintenance mode prior to the shutdown operation. 

1. Go to System Management -> Advanced Options -> Maintenance and click on "Turn Maintenance On" 
2. Set the maintenance duration time and click on "Start Maintenance" 
3. The system will enter into maintenance mode for the configured time. 
4. Once the time is done, the system will leave the maintenance mode automatically. 

* To leave maintenance mode before the configured time duration Click on "Turn maintenance off"
