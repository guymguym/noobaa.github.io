A cloud resource can be either an Azure blob storage, AWS S3 bucket or any AWS S3 compatible service. Cloud resource can be used for NooBaa's bucket data placement policy Learn more about bucket data placement.

Steps:

  1. Go to Resources > Cloud Storage > Add Cloud Resource

  2. In the target connection drop-down select “Add new connection” to create a new public cloud connection access or choose an existing one.

  3. Choose a target bucket you wish to use for storage purposes (this bucket will store the files uploaded to the NooBaa bucket you will decide to connect. This data will only be readable with NooBaa access)

  4. Edit the resource name according to the system names rules - this name cannot be edited later on.

  5. Click “Create” and the new resource will be added to the cloud storage list. This might take a few minutes of initializing the resource.

    If you got an error, please check the following article about cloud resource connection troubleshooting.
