S3cmd configuration might be tricky, but in this article, you will be able to use it in under 5 minutes.

Create a file called noobaa.conf with the following text. don't forget to replace <NooBaa IP>, <NooBaa access key>, <NooBaa secret key> with the real values. You can use DNS name instead of IP. 


[default]
host_base = <NooBaa IP> 
host_bucket = <NooBaa IP> 
access_key = <NooBaa access key> 
secret_key = <NooBaa secret key> 
use_https = False
check_ssl_certificate = False
check_ssl_hostname = False
signature_v2 = True

Now simply run s3cmd command. For example:

s3cmd --config noobaa.conf ls s3://first.bucket
