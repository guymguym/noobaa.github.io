NooBaa Buckets are data containers that behave similarly to AWS S3 buckets. In order to upload data, you can use the first bucket that was automatically created for you on system creation or create a new one. Alternatively, You can create a bucket using the management console or via AWS S3 compatible application.

Steps:

  1. Go to Buckets > Create Bucket

  2. Type a valid bucket name according to the name constraints (S3 protocol restrictions).
  3. On set policy step you need to choose the policy type you wish to apply for this bucket. You can choose either to spread the bucket’s data among the chosen resources or to mirror for full duplication between pools. Learn more about data placement here

  4. Choose the resources you wish to place your data on.
    * If you chose mirror policy, make sure that you have selected at least 2 resources for mirror replication.
    * Combining on-premise and cloud resources may cause performances issues.

  5. Once done, click on “Create”. The new bucket will be added to the bucket list.
