Upon request, you can get NooBaa as a qcow2 format. 
Once you download the disk image, follow the steps below in order to deploy it.

Open the Virtual Machine Manager


Click on "Create a new virtual machine" button. Select the option "Import existing disk image" and click on the Forward button



When choosing a storage volume click on the Browse Local button


Select the downloaded image you got from NooBaa.


Keep the defaults of Generic OS type and Generic Version and click on Forward.



For POC, you can keep the default RAM and CPU or increase it for better performance. Click on Forward.



Name the VM and click Finish.



You are ready to go. Enter the user name and password (in green), set up the network and access the web UI.
