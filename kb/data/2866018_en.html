Network Time Protocol (NTP) is a networking protocol for clock synchronization between computer systems over packet-switched, variable-latency data networks.

Steps:
1. Go to Management > Configuration > Date & Time > Click “edit” on the wanted server
2. On the edit server date & time window, choose the network time option

3. Select the correct time zone of the server
4. Enter the server NTP as IP or DNS name and click on “Save”
