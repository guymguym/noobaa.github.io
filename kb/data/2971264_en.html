When aws cli is needed, you can connect it to your NooBaa system.
The following steps help you to do it.

1. Install aws cli -  https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-install.html
2. run the following command. Provide your credentials by using the following article
     aws configure --profile noobaa

AWS Access Key ID [****************RqzV]: 
AWS Secret Access Key [****************eUFm]: 
Default region name [None]: 
Default output format [None]: 

3.  Test with ls
aws s3 --profile noobaa --endpoint <Replace with your endpoint> ls

Tip: you can run the following command 
alias s3='AWS_ACCESS_KEY_ID=XXXXX AWS_SECRET_ACCESS_KEY=XXXXX aws --endpoint <endpoint> s3'

once you run this command, you can use the commands in a short form:
s3 ls
s3 ls first.bucket
