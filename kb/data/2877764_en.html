When adding a new storage node, you may see it as an offline node, with "No Access" issue (see the image below).

The first step to troubleshooting such case is to check the firewall settings on this node.

Additional information about the ports setup can be found in the following article

Please make sure that the ports range of 60100-60600 is open for inbound traffic. These ports are used to communicate between the storage nodes to allow an efficient self-healing behavior. 

Once you verified that the ports are open, please navigate to the node -> Test & Diagnostics and click on "Run Test" button. This test will run connectivity test and it should pass if the firewall is configured correctly.
