Setting high availability cluster can be done via the cluster menu. Simply click on the "Attach Server" button.
Complete the following steps:
  1. Firewall - The servers should be able to communicate on port 27000
  2. Provision a new server with an OVA that matches the same version you are using.  No need to activate the new server.
  3. Open a console on the new server, login and setup networking and NTP.
  4. Copy the server IP and secret key shown on the console.
  5. Fill the following fields in the cluster setup window.
* Please note that you need at least 3 machines in a cluster.
