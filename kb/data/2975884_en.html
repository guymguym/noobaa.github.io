The first step after the VM deployment is network setup.
Start with the VM settings -> Network -> verify that you are using bridge mode to the right host adapter.



Now start the machine and open the VM console and login with the credentials that you can see in green



Click OK



Select Networking Configuration option



Select Configure interface IP option


Select your network adapter (usually 1)


Choose either static IP or dynamic.



Once the network is set, you can continue to the Getting started article here
