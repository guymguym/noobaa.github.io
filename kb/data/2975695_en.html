NooBaa provides multi-cloud and hybrid cloud capabilities to Ceph, with effortless deployment. 
Navigate to Resources-> Cloud Storage tab. Click on Add Cloud Resource, and select Add new connection.
Simply add a new cloud connection with S3 V4 Compatible service and provide the credentials



Use the newly created connection to your Ceph cluster and map it to a specific bucket


Open data placement configuration of the bucket you want to turn into a hybrid, and select both your Ceph cloud resource and native cloud provider resource and you are done
